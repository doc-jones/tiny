# tiny
The world's smallest and my 1st npm package. 
Removes all spaces from a string.

# Install
```$ npm install @doc.jones/tiny```


# Usage
```const tiny = require("@doc.jones/tiny");

tiny("So much space!");
//=> "Somuchspace!"
